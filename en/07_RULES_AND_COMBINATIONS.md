# 📌 Important Rules for Claude

1. **Execute all phases back-to-back without asking for confirmation** — this is an agentic workflow
2. **Never recommend copying a channel** — always be first in a new market
3. **Prefer 🟢 Untouched over ⚠️ Marginally Used** — a fresh green with medium proof beats a crowded proven market; 🟩 ZERO with proven demand = the best possible state
4. **All proposals must be 100% faceless and doable without showing a face** — only voiceover + animation/AI images/footage — no exceptions
5. **Be specific** — generic advice is useless; give real video titles, thumbnail concepts, and tool recommendations
6. **Validate before recommending** — Phase 3 search validation is mandatory, not optional
7. **Re-rank after validation** — Phase 2 and Phase 3 rankings may differ; use Phase 3 as the final verdict
8. **One competitive advantage per proposal** — identify which the builder can leverage:
   - Professional expertise (dentist → dental channel)
   - Format-first discovery (first to use this format in this market)
   - Cross-industry insight (fitness person discovering a finance format)
   - Production capability (access to 3D, AI tools, unique visual style)
9. **⛔ Gaming and Anime:** Never use these two categories in any stage — discovery, analysis, or bending
10. **Always send the full channel link** — no channel is acceptable without a full URL
11. **Always mention channel descriptions** — extract from `youtube_channel_about` for every channel
12. **Mandatory bilingual visual format** — every English title/channel name must use:
    ```
    🇬🇧 [English text]
    🔤 Translation: [Persian meaning]
    ```
13. **Register today's date in Phase 0** — all 90-day calculations use the real date
14. **YouTube search for each Bend is mandatory** — before presenting any Bend, search YouTube directly
15. **Sub-niche first, then other categories** — explore sub-niches within the same category first
16. **Detect the channel's strategy type** — determine FORMAT TREND / MARKET PLAY / HYBRID before any Bend
17. **Small failed channels gate is mandatory** — lack of big competitors is not enough
18. **Every Niche Bend must have a Complexity Tier label** — Bottom/Middle/Top of Pyramid with budget and time horizon
19. **Phase 3.5 (low-cost testing) is mandatory** — no recommendation without a concrete test plan

## 💡 Powerful Combination Methods

### Niche Transfer
Take proven content to a different market:
- Medieval life → Early humans → Pirates → Homeless
- "Why X is collapsing and you're next" → Civilizations / Marriages / Dictators / Streamers

### Hype Injection
Proven format + celebrity topics = foolproof formula:
- "The Death of Every [X] Explained" → Celebrity version

### Script Bend
Keep proven skeleton, swap topic:
- Keep hook, open loop, payoff structure
- Only change subject to something from the new market

### Twist Injection
Any proven content + unexpected angle + different audience = high CTR

### Geography Bend
Same successful niche + untouched language/regional market = ZERO COMPETITION:
- English Finance → Spanish for Latin market
- American True Crime → European/British crime

### Specificity Bend
Proven general format + narrow sub-audience = higher RPM/ad deals:
- General 3D looping → dedicated gambling/casino version
- General car channel → dedicated F1 version
