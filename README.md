## 📖 Overview | نمای کلی

This repository contains the **YouTube Niche Master Prompt v6** — a comprehensive, multi-phase system prompt designed for **Claude AI** with **NexLev MCP**, **Web Search**, and **Code Execution** tools. It automates the end-to-end process of discovering, analyzing, and validating profitable faceless YouTube channel opportunities.

## 🚀 How to Use | نحوه استفاده

### English
1. Open a **new Claude session** with **NexLev MCP**, **Web Search**, and **Code Execution** enabled
2. Copy the full prompt from [`en/00_MASTER_PROMPT_v6.md`](en/00_MASTER_PROMPT_v6.md) — or combine individual phase files as needed
3. Follow the 6-phase workflow (Phase 0 → Phase 1 → Phase 2 → Phase 3 → Phase 3.5 → Phase 4)
4. Claude will automatically execute all phases without stopping for confirmation

### فارسی
1. یک **سشن جدید Claude** با فعال‌بودن **NexLev MCP**، **Web Search** و **Code Execution** باز کنید
2. پرامپت کامل را از [`YouTube_Niche_Master_Prompt_v6_FA.md`](YouTube_Niche_Master_Prompt_v6_FA.md) کپی کنید
3. یا از فایل‌های تفکیک‌شده در پوشه [`fa/`](fa/) استفاده کنید
4. workflow ۶ فاز را دنبال کنید
5. Claude تمام فازها را **بدون توقف برای تأیید** اجرا می‌کند

## 📋 Workflow Phases | فازهای workflow

| Phase | Name | Purpose |
|-------|------|---------|
| **0** | Setup | Register today's date for freshness calculations |
| **1** | Niche Discovery | Find faceless channels, filter by age/monetization/saturation |
| **2** | Format Analysis & Niche Bending | Analyze channel DNA, generate 3 niche bends across 4 axes |
| **3** | Search Validation | Validate each bend via YouTube & web search |
| **3.5** | Low-Cost Testing | Plan a $100–$200 test before full commitment |
| **4** | Script DNA Extraction | (Optional) Extract script templates from source channel |
