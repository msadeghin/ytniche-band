## گام صفر — ثبت تاریخ امروز (اجباری)

**قبل از هر اقدامی، تاریخ دقیق امروز را از سیستم یا web search دریافت کنید و ثبت کنید:**

```
📅 تاریخ امروز: [YYYY-MM-DD]
📅 مرز ۹۰ روز: [YYYY-MM-DD]   ← تاریخ امروز منهای ۹۰ روز
```

تمام بررسی‌های «۹۰ روز گذشته» در این workflow بر اساس همین دو تاریخ محاسبه می‌شود.
هیچ کانالی را بدون این محاسبه بررسی نکنید.

---

## 🚀 شروع — دریافت ورودی اولیه

قبل از هر کاری، این سؤال را از کاربر بپرسید:

```
سلام! قبل از شروع، به یک پاسخ سریع نیاز دارم تا بهترین آنالیز ممکن رو اجرا کنم:

━━━━━━━━━━━━━━━━━━━━━━━━━━
📌 چه نوع آنالیزی می‌خوای؟
━━━━━━━━━━━━━━━━━━━━━━━━━━

A) 🤖 AUTO — پیدا کردن بهترین فرصت به‌صورت خودکار (هیچ نقطه شروعی نداری)
B) 🔗 CHANNEL — لینک یک کانال داری که می‌خوای فرمتش رو بشکافیم
C) 🎬 VIDEO — لینک یا عنوان یک ویدیوی وایرال داری که می‌خوای تحلیل بشه
D) 🎯 KEYWORD — یک موضوع یا کلیدواژه خاص داری که می‌خوای به نیچ‌های جدید Bend بشه

با A / B / C / D جواب بده، و در صورت نیاز لینک یا توضیح رو هم بفرست.
```
