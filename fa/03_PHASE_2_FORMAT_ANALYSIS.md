## مرحله ۱ — تجزیه فرمت

از `youtube_channel_outliers` + `get_bulk_video_transcripts` روی ۵ ویدیوی برتر استفاده کن.

### استخراج این عناصر

| عنصر | گزینه‌ها |
|------|---------|
| **سبک بصری** | انیمیشن ۲بعدی / ۳بعدی / آواتار AI / talking-head / مستند / screen-record / غیره |
| **ساختار روایی** | لیست / توضیح‌دهنده / داستان / مقایسه / تحول / روز-در-زندگی |
| **مکانیزم هوک** | شکاف کنجکاوی / شوک / قابل‌ارتباط‌بودن / طنز / تریگر هویتی |

| **پیچیدگی تولید** | پایین (ابزارهای AI) / متوسط (انیمیشن ساده) / بالا (۳بعدی/پیچیده) |

## مرحله ۱.۵ — تشخیص نوع استراتژی کانال (اجباری)

مشخص کن کانال منبع از کدام استراتژی استفاده می‌کند:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
🧭 نوع استراتژی کانال
━━━━━━━━━━━━━━━━━━━━━━━━━━
نوع: [FORMAT TREND | MARKET PLAY | HYBRID]

تعریف‌ها:
• FORMAT TREND  → موفقیت کانال بیشتر به فرمت منحصربه‌فردش وابسته است
                  (سبک بصری، ساختار روایی، یا مکانیزم هوک خاص)
                  ▶ استراتژی Bend: بازار (حوزه) را عوض کن، فرمت را نگه دار

• MARKET PLAY   → موفقیت کانال بیشتر به انتخاب بازار/موضوع وابسته است
                  (نیچ داغ، مخاطب پرتقاضا، یا RPM بالا)
                  ▶ استراتژی Bend: فرمت را عوض کن، بازار را نگه دار

• HYBRID        → هم فرمت و هم بازار هر دو نقش کلیدی دارند
                  ▶ استراتژی Bend: هر دو محور را بررسی کن

تشخیص: [توضیح ۱-۲ جمله‌ای]
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### قانون Bend بر اساس نوع استراتژی

| نوع استراتژی | تمرکز در مرحله ۳ |
|--------------|------------------|
| **FORMAT TREND** | تغییر حوزه (فرمت ثابت بماند) |
| **MARKET PLAY** | تغییر فرمت در همان حوزه یا حوزه مشابه |
| **HYBRID** | هر دو محور را بررسی کن |

## مرحله ۲ — آنالیز بازار فعلی

- کدام یک از ۸ دسته‌بندی محتوا (بدون Gaming و Anime) این کانال را ارائه می‌دهد؟
- خواسته اصلی مخاطب / مشکلی که حل می‌شود
- محدوده تخمینی RPM: پایین (<۲$) / متوسط (۲$–۸$) / بالا (۸$–۳۰$+)
- وضعیت اشباع: پیشگام / یکی از معدود‌ها / جمعیت در حال رشد / اشباع‌شده

### سیگنال‌های هشدار اشباع

| سیگنال | اگر موارد زیر بود اشباع علامت بزن |
|--------|-----------------------------------|
| 🔴 | ۵+ کانال دقیقاً همین فرمت را در دقیقاً همین بازار استفاده می‌کنند |
| 🔴 | کانال اصلی بیش از ۱ میلیون ساب دارد و بیشتر از ۱۲ ماه فعال است |
| 🔴 | بخش نظرات می‌گوید «این فقط کپی X است» |

```

• پرتغالی/برزیلی (بازار برزیل)
• آیا کسی این نیچ را به این زبان انجام داده؟ [بله/خیر]
```

## مرحله ۴ — ۳ Niche Bend پیشنهادی برتر

> **رتبه‌بندی با ترکیب:** امتیاز فرصت + اندازه بازار + پتانسیل RPM + قابلیت انتقال فرمت + نتایج جستجوی یوتیوب

### فیلتر نهایی فیسلس

هر Bend پیشنهادی باید این سؤال را پاس کند:
> «آیا این کانال را می‌توان ۱۰۰٪ بدون نشان دادن چهره، فقط با voiceover + انیمیشن/تصاویر AI/footage ساخت؟»
→ اگر جواب خیر است یا مبهم است → این Bend را حذف کن

```
━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 منشأ این Bend:
```
