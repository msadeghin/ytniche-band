## تمپلیت خروجی DNA اسکریپت

```
===================================
تمپلیت اسکریپت [نام کانال]
طول هدف: [X] دقیقه | تعداد payoff هدف: [N]
===================================

هوک [۰:۰۰–۰:۳۰]
نوع: [نوع غالب هوک]
تمپلیت: «[فرمول دقیق ابتدا با اسلات‌های {TOPIC}، {STAT}، {CLAIM}]»

OPEN LOOP شماره ۱
تمپلیت: «[عبارت مشخصه با اسلات {TEASE}]»

PAYOFF ۱ [۰:۳۰–X:XX]
ساختار: [همان‌طور که مشاهده شد — مثلاً ادعا → مثال → داده → نتیجه‌گیری قابل‌اجرا]
تمپلیت:
«[آغازگر بخش با صدای کانال با {POINT_1}]
[الگوی بدنه: {EVIDENCE}، {EXAMPLE}، {INSIGHT}]
[پایان payoff: {ACTIONABLE_LINE}]»

عبارت ترنزیشن: «[کانکتور تکراری دقیق]»

[برای همه N payoff ادامه بده...]

CTA وسط ویدیو: «[تمپلیت دقیق اگر کانال از آن استفاده می‌کند]»

پایان نهایی: «[تحویل اوج‌گیرنده + CTA پایانی]»

===================================
قوانین صدا (غیرقابل‌مذاکره)
===================================
- ریتم جمله: [الگوی مشاهده‌شده]
- سطح خوانایی: [پایه]
- دیدگاه: [اول / دوم / سوم شخص]
- هرگز استفاده نکن: [عبارات غایب از اسکریپت‌های آن‌ها]
- همیشه استفاده کن: [۳–۵ خصوصیت مشخصه شناسایی‌شده]
```

---

## ابعاد تحلیل

| مؤلفه | چه چیزی استخراج شود |
|-------|---------------------|
| **هوک** | نوع، فرمول تمپلیت با متغیرهای اسلات |
| **Open Loops** | عبارت‌های مشخصه، ساختار تیزر |
| **Payoff‌ها** | الگوی ساختار، انواع شواهد، پایان‌های قابل اجرا |
| **ترنزیشن‌ها** | عبارت‌های رابط تکراری |
| **CTAها** | تمپلیت‌های میان ویدیو و پایانی |
| **قوانین صدا** | ریتم، سطح خوانایی، دیدگاه، عبارات ممنوعه/الزامی |
