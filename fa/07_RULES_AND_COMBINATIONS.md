# 📌 قوانین مهم برای Claude

۱. **تمام فازها را پشت‌سرهم و بدون درخواست تأیید اجرا کن** — این یک workflow عاملی است
۲. **هرگز کپی کردن کانال را توصیه نکن** — همیشه اول در بازار جدید باش
۳. **🟢 دست‌نخورده را به ⚠️ تا حدی استفاده‌شده ترجیح بده** — یک سبز تازه با اثبات متوسط بهتر از یک بازار شلوغ اثبات‌شده است؛ 🟩 ZERO با تقاضای اثبات‌شده = بهترین حالت ممکن
۴. **تمام پیشنهادات باید ۱۰۰٪ فیسلس و بدون نشان دادن چهره قابل انجام باشند** — فقط voiceover + انیمیشن/تصاویر AI/footage — هیچ استثنایی وجود ندارد
۵. **مشخص باش** — توصیه‌های کلی بی‌فایده است؛ عنوان‌های ویدیوی واقعی، مفاهیم تامبنیل، و پیشنهادات ابزار بده
۶. **قبل از توصیه، اعتبارسنجی کن** — اعتبارسنجی جستجوی فاز ۳ اجباری است، اختیاری نیست
۷. **بعد از اعتبارسنجی بازرتبه‌بندی کن** — رتبه‌بندی فاز ۲ و فاز ۳ ممکن است متفاوت باشند؛ از فاز ۳ به‌عنوان حکم نهایی استفاده کن
۸. **یک مزیت رقابتی به ازای هر پیشنهاد** — شناسایی کن کدام یک از موارد زیر را سازنده می‌تواند بهره‌برداری کند:
   - تخصص حرفه‌ای (دندانپزشک → کانال دندان)
   - کشف فرمت‌اول (اولین استفاده از این فرمت در این بازار)
   - بینش بین‌صنعتی (شخص فیتنس که یک فرمت مالی را کشف می‌کند)
   - قابلیت تولید (دسترسی به ۳بعدی، ابزارهای AI، سبک بصری منحصربه‌فرد)
۹. **⛔ Gaming و Anime:** از این دو دسته‌بندی در هیچ مرحله‌ای استفاده نکن
۱۰. **لینک کامل هر کانال را همیشه ارسال کن** — هیچ کانالی بدون URL کامل قابل‌قبول نیست
۱۱. **توضیحات کانال را همیشه ذکر کن** — از `youtube_channel_about` برای هر کانال استخراج و درج کن
۱۲. **فرمت بصری زبان اجباری است** — هر عنوان ویدیو، نام کانال، یا متن انگلیسی باید با این فرمت نوشته شود:
    ```
    🇬🇧 [متن انگلیسی]
    🔤 ترجمه: [معنی فارسی]
    ```
۱۳. **تاریخ امروز را در فاز ۰ ثبت کن** — تمام محاسبات ۹۰ روز بر اساس تاریخ واقعی باشد
۱۴. **جستجوی یوتیوب برای هر Bend اجباری است** — قبل از معرفی هر Bend، مستقیماً در یوتیوب جستجو کن
۱۵. **ابتدا زیرحوزه، سپس حوزه دیگر** — قبل از Bend به حوزه جدید، زیرحوزه‌های داخل همان حوزه را بررسی کن
۱۶. **نوع استراتژی کانال را تشخیص بده** — قبل از هر Bend، FORMAT TREND / MARKET PLAY / HYBRID را مشخص کن
۱۷. **دروازه‌ی کانال‌های کوچک شکست‌خورده اجباری است** — نبود رقیب بزرگ کافی نیست
۱۸. **هر Niche Bend باید برچسب لایه‌ی پیچیدگی داشته باشد** — پایین/وسط/بالای هرم با بودجه و افق زمانی
۱۹. **فاز ۳.۵ (تست ارزان) برای پیشنهاد نهایی اجباری است** — هیچ توصیه‌ای بدون پلن تست عملی ارائه نشود

## 💡 روش‌های ترکیب قدرتمند

### انتقال نیچ
محتوای اثبات‌شده را به بازار متفاوتی ببر:
- زندگی قرون وسطی → انسان‌های اولیه → دزدان دریایی → بی‌خانمان‌ها
- «چرا X فرو می‌پاشد و چرا تو بعدی هستی» → تمدن‌ها / ازدواج‌ها / دیکتاتورها / استریمرها

### تزریق هایپ
فرمت اثبات‌شده + موضوعات محور سلبریتی = فرمول ضداشتباه:
- «مرگ هر [X] توضیح‌داده‌شده» → نسخه سلبریتی

### Bend اسکریپت
اسکلت اثبات‌شده + ایده تازه از نیچ خودت:
- هوک، open loop و ساختار payoff را نگه دار
- فقط موضوع را با چیزی از بازار جدید عوض کن

### تزریق پیچ
هر محتوای اثبات‌شده + زاویه غیرمنتظره + مخاطب متفاوت = CTR بالا

### Bend جغرافیایی
همان نیچ موفق + بازار زبانی/منطقه‌ای دست‌نخورده = ZERO COMPETITION:
- کانال موفق Finance انگلیسی → نسخه اسپانیایی برای بازار لاتین
- کانال موفق True Crime آمریکایی → نسخه متمرکز بر جرایم اروپایی/بریتانیایی

### Bend اختصاصی‌سازی (Specificity Bend)
فرمت عمومی اثبات‌شده + محدودکردن به یک زیرمخاطب مشخص = مخاطب وفادارتر + RPM بالاتر:
- کانال عمومی لوپلی سه‌بعدی → نسخه‌ی اختصاصی قمار/کازینو
- کانال عمومی خودرو → نسخه‌ی اختصاصی F1
